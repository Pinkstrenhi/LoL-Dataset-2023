# League of Legends Match Data (2023)

This repository provides a partial sample of a larger dataset of *League of Legends* matches collected via the Riot Games API.

## Dataset Description

The dataset contains matches played between January and October 2023. Data collection was conducted from October 2023 to February 2024.

Due to GitHub file size limitations and the constraints of anonymous repository submission, only a reduced sample of the dataset is included.

## Data Variability

The sample includes data from three regions:

* AMERICAS (BR1)
* ASIA (KR)
* SEA (TW2)

Matches from all player-roles (Top, Jungle, Mid, ADC, and Support) are included, covering both wins and losses.

## Anonymization

Player usernames were anonymized using a "Player_Number" convention, where the number corresponds to the ordinal position of the player's first appearance in the dataset.

## Full Dataset Availability

The complete dataset comprises approximately one million instances, each representing a single player in a match. In total, it includes 98,138 unique matches, each involving 10 players.

The full dataset will be made publicly available upon acceptance of the study:

**"Beyond KDA: A Strategy-Aware Approach to Performance Measurement and Outcome Prediction in MOBA Games"**
